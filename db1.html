DEBER 1 - SENTENCIA IF

La sentencia if en Python es una estructura de control condicional que ejecuta un bloque de código solo si una condición específica es verdadera.

Resuelve los siguientes ejercicios usando condicionales en Python. Estos ejercicios aplican estructuras condicionales (if, elif, else) en Python para resolver problemas básicos como validaciones, comparaciones y toma de decisiones.

1. Número mayor a 50


# Solicita al usuario un número entero
numero = int(input("Ingrese un número: "))

# Verifica si el número es mayor que 50
if numero > 50:
    # Si se cumple la condición, muestra el mensaje
    print("El número es mayor que 50")

2. Puede votar


# Solicita la edad del usuario
edad = int(input("Ingrese su edad: "))

# Evalúa si es mayor o igual a 18
if edad >= 18:
    # Si es mayor de edad, puede votar
    print("Puede votar")
else:
    # Caso contrario, no puede votar
    print("No puede votar")

3. Múltiplo de 3


# Solicita un número al usuario
numero = int(input("Ingrese un número: "))

# Verifica si el número es divisible para 3
# El operador % devuelve el residuo
if numero % 3 == 0:
    print("Es múltiplo de 3")
else:
    print("No es múltiplo de 3")

4. Contraseña


# Solicita una contraseña al usuario
clave = input("Ingrese la contraseña: ")

# Compara si la contraseña ingresada es correcta
if clave == "python123":
    print("Correcta")
else:
    print("Incorrecta")

5. Nota


# Solicita una nota (número decimal)
nota = float(input("Ingrese la nota: "))

# Evalúa si la nota es suficiente para aprobar
if nota >= 7:
    print("Aprobado")
else:
    print("Reprobado")

6. Número mayor


# Solicita dos números enteros
a = int(input("Número 1: "))
b = int(input("Número 2: "))

# Compara cuál número es mayor
if a > b:
    print("El primero es mayor")
elif b > a:
    print("El segundo es mayor")
else:
    # Si ninguno es mayor, son iguales
    print("Son iguales")

7. Positivo, negativo o cero


# Solicita un número al usuario
num = int(input("Ingrese un número: "))

# Determina si el número es positivo, negativo o cero
if num > 0:
    print("Positivo")
elif num < 0:
    print("Negativo")
else:
    print("Cero")

8. Vocal


# Solicita una letra y la convierte a minúscula
letra = input("Ingrese una letra: ").lower()

# Verifica si la letra está dentro del grupo de vocales
if letra in "aeiou":
    print("Es vocal")
else:
    print("No es vocal")

9. Temperatura


# Solicita la temperatura
temp = float(input("Ingrese temperatura: "))

# Evalúa si la temperatura es menor a 15 grados
if temp < 15:
    print("Hace frío")
else:
    print("No hace frío")

10. Año bisiesto


# Solicita un año al usuario
año = int(input("Ingrese un año: "))

# Verifica si es un año bisiesto:
# - Debe ser divisible para 4
# - No debe ser divisible para 100
# - Excepto si es divisible para 400
if (año % 4 == 0 and año % 100 != 0) or (año % 400 == 0):
    print("Es bisiesto")
else:
    print("No es bisiesto")
