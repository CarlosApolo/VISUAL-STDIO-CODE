DEBER 2 - BUCLES

Los bucles en Python permiten repetir un bloque de código varias veces, ya sea mediante estructuras for o while.

Aplica estructuras de repetición (for / while) para resolver los siguientes problemas. Comenta el código de cada solución.

1. Imprime los números del 1 al 10.


# El ciclo for recorre un rango de números
# range(1, 11) genera números desde 1 hasta 10
for i in range(1, 11):
    # En cada iteración se imprime el valor de i
    print(i)

2. Imprime los números pares del 1 al 20.


# Recorre los números del 1 al 20
for i in range(1, 21):
    # Verifica si el número es par usando el operador módulo
    if i % 2 == 0:
        print(i)

3. Tabla de multiplicar.


# Solicita un número al usuario
numero = int(input("Ingrese un número: "))

# Recorre del 1 al 10
for i in range(1, 11):
    # Muestra la multiplicación paso a paso
    print(numero, "x", i, "=", numero * i)

4. Números del 10 al 1.


# range(inicio, fin, paso negativo)
# Aquí empieza en 10 y va hasta 1 restando de uno en uno
for i in range(10, 0, -1):
    print(i)

5. Suma acumulada.


# Solicita un número
numero = int(input("Ingrese un número: "))

# Variable acumuladora inicializada en 0
suma = 0

# Recorre desde 1 hasta el número ingresado
for i in range(1, numero + 1):
    # Se va sumando cada valor a la variable suma
    suma += i

# Muestra el resultado final
print("La suma es:", suma)

6. Contar vocales.


# Solicita una palabra y la convierte a minúsculas
palabra = input("Ingrese una palabra: ").lower()

# Contador inicial en 0
contador = 0

# Recorre cada letra de la palabra
for letra in palabra:
    # Verifica si la letra es una vocal
    if letra in "aeiou":
        contador += 1  # Incrementa el contador

# Muestra el total de vocales
print("Número de vocales:", contador)

7. Validar contraseña.


# Contraseña correcta definida
correcta = "python123"

# Bucle infinito hasta que se cumpla la condición
while True:
    # Solicita la contraseña
    clave = input("Ingrese la contraseña: ")

    # Verifica si coincide con la correcta
    if clave == correcta:
        print("Acceso correcto")
        break  # Sale del ciclo si es correcta
    else:
        print("Contraseña incorrecta")

8. Múltiplos de 5 del 1 al 50.


# Recorre los números del 1 al 50
for i in range(1, 51):
    # Verifica si el número es divisible para 5
    if i % 5 == 0:
        print(i)

9. Número mayor.


# Inicializa la variable mayor como None (sin valor inicial)
mayor = None

# Se repite 5 veces
for i in range(5):
    # Solicita un número
    num = int(input("Ingrese un número: "))

    # Si es el primer número o es mayor al actual
    if mayor is None or num > mayor:
        mayor = num  # Se actualiza el valor mayor

# Muestra el número más grande ingresado
print("El número mayor es:", mayor)

10. Triángulo de asteriscos.


# Recorre del 1 al 5
for i in range(1, 6):
    # Imprime una cantidad de asteriscos igual a i
    # El operador * repite el carácter
    print("*" * i)
