DEBER 3 - FUNCIONES

Las funciones en Python son bloques de código reutilizables que realizan una tarea específica. Permiten organizar mejor el programa y evitar repetir código.

Crea funciones en Python que resuelvan cada uno de los siguientes problemas. Recuerda comentar cada función indicando su propósito, parámetros y valor de retorno.

1. Mensaje de bienvenida.


def bienvenida(nombre):
    print("Bienvenido,", nombre)

bienvenida("Carlos")

2. Número par o impar.


def es_par(numero):
    if numero % 2 == 0:
        return "Par"
    else:
        return "Impar"

print(es_par(5))

3. Suma de dos números.


def suma(a, b):
    return a + b

print(suma(3, 4))

4. Longitud de palabra.


def longitud(palabra):
    return len(palabra)

print(longitud("Python"))

5. Cuadrado de un número.


def cuadrado(numero):
    return numero ** 2

print(cuadrado(4))

6. Mayor de tres números.


def mayor(a, b, c):
    if a >= b and a >= c:
        return a
    elif b >= a and b >= c:
        return b
    else:
        return c

print(mayor(5, 9, 3))

7. Celsius a Fahrenheit.


def celsius_a_fahrenheit(c):
    return (c * 9/5) + 32

print(celsius_a_fahrenheit(25))

8. Suma de lista.


def suma_lista(lista):
    total = 0
    for num in lista:
        total += num
    return total

print(suma_lista([1, 2, 3, 4]))

9. Número positivo.


def es_positivo(numero):
    if numero > 0:
        return True
    else:
        return False

print(es_positivo(10))

10. Invertir palabra.


def invertir(palabra):
    return palabra[::-1]

print(invertir("python"))
